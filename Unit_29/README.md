## 29.2 덧셈 함수 만들기

함수에서 값을 받으려면 ```( )``` 안에 변수 이름을 지정해주면 됩니다. 특히 이 변수를 매개변수(parameter)라고 부릅니다.
```
def 함수이름(매개변수1, 매개변수2):
    코드
```
